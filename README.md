# Iqra App

Interactive Quran Learning Application built with React, TypeScript, and Vite.

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- Framer Motion

## Getting Started

### Prerequisites

- Node.js (recommended: use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))
- npm or yarn

### Installation

```sh
# Clone the repository
git clone https://github.com/sgizemmazi/iqra-app.git

# Navigate to the project directory
cd iqra-app

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build for development
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm test` - Run tests
- `npm run test:watch` - Run tests in watch mode
